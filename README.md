# BudgetingCalc

# Overview

The goal of this project was to get familiar with C++ as well as making a program utilizing a graphical user interface.

This project allows you to input monthly income and expense amounts and then outputs a total for you to view.


[Software Demo Video](https://youtu.be/EEcWY3M02ko)

# Development Environment


This program was made in Visual Studio.

This project was created using C++.
To make the GUI I utilized windows MyForm.
